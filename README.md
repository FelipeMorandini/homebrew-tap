# Homebrew Tap

Homebrew formulae for tools by [@felipemorandini](https://github.com/felipemorandini).

## Installation

```bash
brew tap felipemorandini/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [smartlog](Formula/smartlog.rb) | A high-performance TUI for log tailing with JSON auto-detection and real-time filtering |
| [jwt-term](Formula/jwt-term.rb) | A secure, offline-first CLI for inspecting and validating JWTs |

### Install

```bash
brew install felipemorandini/tap/smartlog
brew install felipemorandini/tap/jwt-term
```

### Update

```bash
brew update
brew upgrade smartlog
brew upgrade jwt-term
```
